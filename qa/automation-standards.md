## Automation Standards

Automation must be:

* Stable
* Fast
* Repeatable
* Deterministic
* Maintainable

## Automation Priority

Automate:

1. Critical User Flows
2. Business Critical Scenarios
3. Regression Scenarios
4. API Contracts
5. Smoke Tests

## Avoid

* Flaky Tests
* Hardcoded Data
* Environment Coupling
* Low Value Automation
* Excessive UI Automation
