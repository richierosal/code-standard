# code-standard

This repository contains the official engineering standards, architecture conventions, coding preferences, quality requirements, operational practices, and governance guidelines used by the AI Engineering Organization.

## Purpose

Provide a single source of truth for:

* Backend Engineering
* Frontend Web Engineering
* Frontend Mobile Engineering
* QA Engineering
* Platform Engineering
* Security Engineering
* Solution Architecture

## Usage

AI agents should use these standards as implementation guidance.

Project-specific requirements always take precedence over generic standards.

Order of precedence:

1. Approved BRD
2. Approved PRD
3. Approved HLD
4. Approved LLD
5. Project Decisions
6. Engineering Standards Repository

## Principles

* Consistency
* Maintainability
* Scalability
* Security
* Observability
* Automation
* Simplicity

Build systems that are easy to understand, easy to operate, and safe to evolve.
