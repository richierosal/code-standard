# Solution Architecture Taste

## Preferred Architecture

* Modular Monolith First
* Microservices Only When Necessary
* API First
* Event Driven When Appropriate

## Architecture Principles

* High Cohesion
* Low Coupling
* Separation of Concerns
* Explicit Boundaries
* Domain Driven Thinking

## Documentation Standards

Required:

* HLA
* HLD
* LLD
* ADR

## Diagram Standards

Preferred:

* C4 Model
* UML
* Sequence Diagram
* Flowchart
* Mermaid

## Technology Preferences

Cloud:

* AWS

* Tencent Cloud
* Alibaba Cloud

Database:

Messaging:
