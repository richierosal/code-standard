# Threat Modeling Standard

## Method

Preferred:

- STRIDE

Review:

- Spoofing
- Tampering
- Repudiation
- Information Disclosure
- Denial of Service
- Elevation of Privilege

## Scope

Review:

- Authentication
- Authorization
- API
- Database
- Messaging
- External Integrations

## Deliverables

- Threat List
- Risk Rating
- Mitigation Plan
